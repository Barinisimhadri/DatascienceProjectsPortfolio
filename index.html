Fraudulent
Transaction Detection

Utilizing machine learning techniques like Isolation Forest and Local Outlier Factor (LOF), this project aims to detect fraudulent transactions in the banking/payment industry in real-time. Isolation Forest, an unsupervised algorithm, identifies anomalies based on the 'few and different' principle using randomly sub-sampled data processed in tree structures. Meanwhile, LOF, another unsupervised algorithm, identifies outliers by evaluating local density, focusing on points deviating from local clusters rather than the entire data distribution in order to enhance fraud detection measures.
